clock API.

Description

clock API. This is a set of functions to create a clock process. clock process run on x interval of time and process schedule tasks

Function Index

add/1 the same as add(ClockName, 120).
add/2 creates a new clock process, the name must be unique and used for further operations.
get/1 Get current clock state.
remove/1 remove a clock process, stopping all the scheduled functions.
sched/2 schedule a function in a clock process, this will run each beat of time, name reference is obtained by function name.
sched/3 schedule a function in a clock process, this will run each beat of time, name reference necesary for unschedule.
set_bpm/2 sets bpm for a clock process.
start/1 set start status to a clock process, count will continue where it was left.
stop/1 set stop status to a clock process, count will retain its value.
unsched/2 unschedule a function in a clock process, you need to schedule it again if you want to use it.
